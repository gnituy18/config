## MacOS Setup

1. 熱點: 右上->通知中心
1. 設定Dock
1. `xcode-select --install`
1. Github SSH key pair
1. 下載 Chrome
1. 下載 Google Drive
1. 下載 Alacritty & terminfo.info
1. 下載 Roboto Mono
1. 下載 brew
1. 下載 nvm
1. 下載 golang
1. 下載 rust
1. 下載 packer
1. `git clone git@github.com:gnituy18/config.git`
1. `cd config && ./setup.sh`
